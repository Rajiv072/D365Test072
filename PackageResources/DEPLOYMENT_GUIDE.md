# Dynamics 365 Finance and Operations Deployment Package

## Package Contents

This deployment package contains:

- **Metadata**: Model files including the HelloWorld class
- **Descriptor**: Package manifest and model descriptors
- **PackageResources**: Supporting resources for deployment

## Installation Steps

1. Extract the package files
2. Import the model into your Dynamics 365 environment using Model Management
3. Build and compile the model
4. Synchronize the database
5. Run the HelloWorld class to verify deployment
