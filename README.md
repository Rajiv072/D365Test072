# Dynamics 365 Finance and Operations Project

This is a sample project for Dynamics 365 Finance and Operations development using X++.

## Getting Started

1. Install the Dynamics AX (Dynamics 365 Finance and Operations) X++ extension in VS Code.
2. Open the project in VS Code.
3. The src folder contains X++ source files.

Note: Full development is typically done in Visual Studio with the D365FO development tools. This VS Code setup provides basic X++ syntax support.
